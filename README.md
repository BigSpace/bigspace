# bigspace

Software stack:
  1. Pure.css - style guide
  2. Angular.js - frontend framework
  3. Express.js - backend framework
  4. mongoose(mongo) - database

To get app running:
  1. clone repo
  2. run npm install
  3. run npm start (in the app dir)
